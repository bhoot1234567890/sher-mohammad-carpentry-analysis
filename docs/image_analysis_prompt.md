# Image Analysis Prompt for Carpenter Portfolio

## Objective
Analyze each carpentry portfolio image to extract detailed information for website display. Focus on technical craftsmanship, materials, techniques, and aesthetic qualities.

## Analysis Criteria

### 1. Basic Identification
- **Image Type**: Close-up detail, full project view, work-in-progress, before/after, tool/equipment, material showcase
- **Primary Subject**: Furniture type, architectural element, tool, material, technique demonstration
- **Estimated Completion Stage**: Rough cut, assembly, sanding, finishing, completed

### 2. Technical Details
- **Materials Identified**: 
  - Wood types (oak, pine, mahogany, walnut, etc.)
  - Hardware (hinges, screws, nails, brackets)
  - Finishes (stain, varnish, paint, oil)
  - Special materials (metal accents, glass, stone inlays)

- **Joinery Techniques**:
  - Dovetail joints
  - Mortise and tenon
  - Finger joints
  - Pocket holes
  - Dowel joints
  - Biscuit joints
  - Lap joints
  - Miter joints

- **Construction Methods**:
  - Frame and panel
  - Solid wood construction
  - Veneer work
  - Bent lamination
  - Carving techniques

### 3. Craftsmanship Quality
- **Precision**: Alignment, symmetry, gap consistency
- **Surface Quality**: Sanding smoothness, finish application
- **Attention to Detail**: Edge treatment, corner perfection, hardware alignment
- **Structural Integrity**: Visible reinforcement, load-bearing design

### 4. Design Elements
- **Style**: Traditional, modern, rustic, industrial, Scandinavian, mid-century
- **Functional Features**: Drawers, shelves, hinges, sliding mechanisms
- **Aesthetic Features**: Inlays, carvings, decorative moldings, grain patterns
- **Color and Finish**: Stain color, paint technique, natural wood preservation

### 5. Project Context (if discernible)
- **Project Type**: Custom furniture, cabinetry, flooring, doors, windows, outdoor structures
- **Scale**: Small decorative item, medium furniture piece, large architectural element
- **Intended Use**: Residential, commercial, decorative, functional
- **Customization Elements**: Client-specific requests, unique design solutions

### 6. Tool and Equipment Analysis (if visible)
- **Hand Tools**: Chisels, planes, saws, clamps
- **Power Tools**: Table saws, routers, planers, sanders
- **Specialty Tools**: CNC equipment, laser guides, jigs
- **Safety Equipment**: Goggles, gloves, dust collection

### 7. Workflow and Process Insights
- **Work Stage Captured**: Measurement, cutting, assembly, finishing, installation
- **Workshop Organization**: Tool layout, material storage, workspace efficiency
- **Quality Control Methods**: Measurement tools visible, test fits, mock-ups

### 8. Aesthetic Evaluation
- **Visual Appeal**: Balance, proportion, harmony of elements
- **Craftsmanship Showcase**: Areas highlighting skill and expertise
- **Unique Selling Points**: What makes this piece stand out
- **Potential Client Appeal**: Target audience for this style/quality

### 9. Technical Challenges (inferred)
- **Complex Joinery**: Difficult angles or connections
- **Material Challenges**: Working with difficult grains or materials
- **Precision Requirements**: Tight tolerances, complex measurements
- **Design Complexity**: Intricate patterns or structural demands

### 10. Portfolio Presentation Suggestions
- **Best Use on Website**: Hero image, detail showcase, process documentation
- **Accompanying Text Suggestions**: Key selling points to highlight
- **Image Cropping/Focus Recommendations**: Areas to emphasize
- **Before/After Potential**: If part of a transformation series

## Output Format Requirements

For each image, provide structured JSON with:
- filename
- timestamp
- analysis (detailed breakdown using above criteria)
- keywords (for SEO and categorization)
- recommended_usage (how to best display on portfolio)
- quality_rating (1-10 scale for craftsmanship visibility)
